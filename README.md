## üìñ How It Works



1.  **Selection:** Choose between Caesar, ROT13, or Base64 from the dropdown. 
2.  **Input:** Paste your text into the input field. The **Character Counter** updates instantly.
3.  **Logic:** The JavaScript engine processes the string through the selected algorithm:
    * **Caesar/ROT13:** Mathematical character shifting with alphabet wrapping.
    * **Base64:** Binary-to-text encoding using native browser methods.
4.  **Export:** Use the "Copy" button to save the result to your clipboard instantly.
